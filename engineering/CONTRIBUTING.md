Contributing to TXLOGPAY

Overview

TXLOGPAY is an operational finance infrastructure project focused on:

- operational workflows
- treasury orchestration
- multicurrency settlement
- programmable finance infrastructure

We welcome contributors aligned with:

- enterprise engineering principles
- operational clarity
- infrastructure scalability
- respectful collaboration

---

Core Engineering Principles

1. Operational-First Thinking

Always prioritize:

- operational workflows
- treasury consistency
- lifecycle clarity

before:

- blockchain exposure
- crypto-native interaction

---

2. Invisible Blockchain

Blockchain should remain:

- infrastructure
  NOT:
- UX centerpiece

Users should never need to:

- manage wallets
- understand blockchain mechanics
- interact with cryptographic complexity

---

3. Service-Centric Architecture

Business logic belongs primarily in:

- services
- domain models
- infrastructure layers

Avoid placing:

- financial logic
- treasury logic
- settlement orchestration

inside:

- React components

---

4. Treasury Consistency

Protected operational capital must remain:

- isolated
- auditable
- consistent

Platform fees must remain:

- separated from protected amounts
